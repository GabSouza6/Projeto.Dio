As tags mais usadas no HTML para desenvolver um site são:

  • h1 até o h6 serve para títulos e subtítulos

  • Tag <strong> Deixar o texto em negrito

Listas Ordenadas

Para poder criar uma lista ordenada, utilizamos a tag <ol>, logo em seguida é necessário utilizar a tag <li> para adicionar os itens na lista

  1. Item 1
  2. Item 2
  3. Item 3

Listas não ordenadas

Para poder criar uma lista não ordenada, utilizamos a tag <ul>, logo em seguida é necessário utilizar a tag <li> para adicionar os itens na lista

  • Item 1
  • Item 2
  • Item 3

As Tags Sup e Sub

Tag Sup

Utlizada para elevar números ou letras, geralmente utilizados no ramo da matemática

Ex: a2 + b2 = c2 , não utilizando a tag <sup> ficaria desse jeito

Após utlizar a mesma, fica desse jeito, Ex: a2 + b2 = c2

Tag Sub

Utilizada para descer números ou letras, geralmente usa mais para números

Ex:H2O

Após utlizar a mesma, observe que o número 2 foi para baixo, H2O
